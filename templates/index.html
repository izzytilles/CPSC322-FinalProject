About This Model

This Random Forest classifier predicts Near Earth Object classifications based on four key features:

  • Estimated Diameter (Min & Max): The size range of the NEO
  • Relative Velocity: How fast the NEO is moving relative to Earth
  • Miss Distance: How close the NEO comes to Earth

Note: All values should be pre-discretized according to the model's binning scheme.
